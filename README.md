## Run Application

## Prerequisites

```
PHP v5.5+
node v4+
Selenium server standalone v2.48+
Firefox (latest)
```

## Installation

```
# this will also run "npm i" to install node deps
composer install
```

## Testing

```
# run all tests
composer test

# run php unit tests
composer test:server
```

## Run Server (locally)
```
PORT=3000 composer start
```
